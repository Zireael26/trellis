# Recipe manifest

One row per recipe. Recipes are generic, parametric skeletons — specifics enter through `args`, never as baked literals. The capability and degrade columns echo the two-level graceful degrade in `SKILL.md`. The **Loop safety** column records each recipe's `meta.safety` declaration — every recipe is a loop and must honor the loop-safety contract (`core-rules/loop-safety.md`); **a recipe with no `safety` declaration is non-compliant** (flagged by `cross-project-process-audit`, a `process-gate` / review finding).
